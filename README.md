Dashboard de Análisis Comercial - Sano y Fresco.
Este repositorio contiene un proyecto integral de Inteligencia de Negocios desarrollado en Power BI. El objetivo principal es transformar datos brutos de ventas y operaciones en información estratégica para la toma de decisiones en el sector retail/consumo masivo ("Sano y Fresco").

🎯 Objetivos del Proyecto
Monitoreo de KPIs: Seguimiento en tiempo real de ventas totales, márgenes de utilidad y volumen de pedidos.

Análisis de Relaciones: Utilización de visualizaciones avanzadas (ForceGraph) para entender la interconexión entre productos, clientes y territorios.

Optimización Operativa: Identificar patrones de compra y segmentación de mercado para mejorar la eficiencia comercial.

🛠️ Tecnologías y Herramientas
Power BI Desktop: Modelado de datos y visualización.

Power Query (M): Limpieza, transformación y estructuración de tablas (ETL).

DAX (Data Analysis Expressions): Creación de medidas dinámicas como variaciones interanuales, acumulados (YTD) y ratios de rentabilidad.

Custom Visuals: Implementación de grafos de red para análisis de dependencias de datos.

📈 Funcionalidades Clave
Segmentación Interactiva: Filtros cruzados por fecha, categoría y vendedor que actualizan todo el reporte instantáneamente.

Diseño UX/UI: Interfaz limpia con navegación intuitiva mediante botones y paneles de filtros ocultables.

Análisis de Redes: Un panel dedicado a visualizar cómo se relacionan las diferentes entidades del modelo de negocio.
